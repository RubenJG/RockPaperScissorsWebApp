About myself and this application

I'm a computer science engineer who graduated from the University of Costa Rica. Ever since I've been mostly a professor with some development experience in a few varied projects. Now I would like to become a software developer, improve my skills and learn new technologies and work methodologies. I enjoy working with good documented frameworks that change the way you look at software.

For this solution I used the following technologies

  • Java in pair with Maven and Spring Boot. I used these technologies since I'm applying to a Java position. I also decided to use this project to learn a little bit about Spring, which is a technology I haven't use before. But there's still a vast amount of Spring to learn.
  • Thymeleaf templates, HTML5, CSS, SASS, Javascript, jQuery. I used these technologies for the frontend of the application. Some don't need much explanation. SASS is a CSS preprocessor I like to use. Thymeleaf template system seems to be Spring's default.
  • JPA. I learned a little bit of this for the data layer of the application. It took care of most CRUD operations.
  • MySQL is my choice of database, since it's free.
  • Tomcat is the container of this application, it is Spring Boot's default.
