Help

In order to play this game you'll need to create a file containing the game or games you want to simulate.

If you want to play only one game, then you'll create an array of 2 players. Each players is also an array that consists in 2 strings, name and strategy. A player's strategy is "R" for Rock, "P" for Paper and "S" for Scissors. Use double quotes when writing your strings.

If you want to play a tournament, you can start creating arrays of games and nest them as much as you would like as long as the amount of players remains a power of two.
